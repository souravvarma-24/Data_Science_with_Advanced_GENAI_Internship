## ‚ñ∂Ô∏è How the Application Works

The Flask application runs on a local development server. The user provides a name using a URL query parameter. Flask retrieves this value using the request object, processes the input by converting it into uppercase using Python string methods, and returns the result as a browser response.

## ‚ñ∂Ô∏è How to Run the Application

First, ensure that Python is installed on your system. Install Flask by running the following command in the terminal or command prompt:

After installing Flask, navigate to the project directory and start the application using:

Once the server starts, open a web browser and use the following URL format:  
http://127.0.0.1:5000/?name=yourname
